###再帰呼び出し
・概要
    自分自身を呼び出して大きな問題を小さく分割しながら解決する

・使い方
    ・階上計算
    ・ツリー探索
    ・探索アルゴリズム，クイックソート

・例/計算方法
    ・階上
        int factrial(int n){
            if(n==0){
                return 1;
            }else{
                return n*factrial(n-1);
            }
        }
    ・ツリー探索：各ノードで同じ処理を子ノードへ再帰的に実施
